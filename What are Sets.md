We are often concerned with whether an object is a member of a certain collection of objects. Generally there are certain properties that the members of this collection may have in common. The collection is defined by that property. We then test whether the objects satisfy this property to check whether they belong to the collection or not.

***Example:*** we might be interested in whether a certain natural number belongs to the collection of even numbers. The property that defines the collection of even natural numbers is whether the number is divisible by 2 or not. To check whether this natural number belongs to this collection we can then check whether it is divisible by 2.

Sets are used to model this concept in mathematics. Informally a set is a collection of objects that share a property. Intuitively we can define virtually any set as long as we have a property that we can use to check for membership. 

Often we might be interested in knowing whether a collection is a member of another collection. If the first collection satisfies the properties of the second then we can meaningfully talk about the first collection being a member of the second. 

***Example:*** We might be interested in finding intervals that satisfy the inequality: $x^2 - 4x + 3 > 0$. The solution to this would be a collection of 2 intervals $(-\infty, 1)$ and $(3, \infty)$. These intervals themselves are collections of real numbers.

### Set Theory

Most mathematical objects could be very conveniently described as sets. From obvious examples like number systems sets of solutions to equations to more unintuitive examples like sequences (due to order of elements mattering on top of just membership, see Ordered N-tuples), functions and relations.
