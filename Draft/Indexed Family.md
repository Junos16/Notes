### Indexing vs Ordering
When we are interested in the order of elements in a collection we model them using Ordered N-Tuples in set theory. The set has a notion of first element (and last in case of a finite tuple), and each element in the tuple has a notion of predecessor and successor associated with it.

But sometimes we might not be interested in the order of the elements but rather use some other notion to organize them. A common way this is done is by associating each element of the collection with another element of a collection called the index. The second set is the indexing set
