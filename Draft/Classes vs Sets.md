### Sets in ZFC
A way to resolve Russel's Paradox was by weakening the Axiom of Universal Comprehension. In ZFC the Axiom Schema of Restricted Comprehension only allows for the existence of sets that are subsets of pre-existing sets. A consequence of this was that we could no longer define sets as "large" (intuitively) as that in Russel's Paradox. 

But a downside to this was that we have to discard these large sets. While most of common mathematics could be defined neatly within this axiomatic system, there are scenarios where it might be useful to use these sets.

### Sets in NBG
Another way to deal with large sets like the Russel's Set is to distinguish between "large" sets like in Russel's Paradox and regular sets used in mathematics. In NBG the Axiom of Class Comprehension allows for the existence of any possible collection of elements but specifies that the collection is not a set. Instead it is a considered a "Class".

Any class that is an element of another class is considered a set. 

- **Set (NBG Set Theory):**  $A\space is\space a\space set \iff \exists C(A \in C)$

Similar to ZFC, most mathematical objects can be defined using these "smaller" collections, sets. Sets that are as "large" as that in Russel's Paradox are considered classes that are not sets. They are considered "Proper Classes"

### Classes
All collections discussed in NBG set theory are called "Classes". This includes all regular mathematical objects called "sets" as well objects that are too "large" to be called sets, i.e. Proper Classes.
