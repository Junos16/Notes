# Vector Spaces?
- **Definition:** A vector space $(V,+,\cdot)$ over a field $F$ is a set $V$ with binary operation $+$ and binary function $\cdot$ such that the following axioms hold
	- **Axioms:**
		1. **Commutativity of Addition:**
		2. **Associativity of Addition:**
		3. **Additive Identity:**
		4. **Additive Inverse:**
		5. **Multiplicative Identity:**
		6. **Distributive Property**
	- **Addition Operation:** $+:V\times V\to V$ 
	- **Scalar Multiplication Operation:** $\cdot:F\times V\to V$
