# Representation 
- Due to the fact that arrays are the most data structures in the context of computers, despite their non-linearity graphs are still implemented using arrays just like every other data structure
- This naturally comes at a cost of simple graph primitive queries needing to be computed using non-constant run-time
- There are two commonly used representation of graphs in the context of computers: 
	1. Using an adjacency matrix
	2. Using an adjacency list
