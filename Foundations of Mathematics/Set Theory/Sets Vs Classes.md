### Need for distinction
- Classes and sets distinguished to avoid logical paradoxes in set theory such as [Russel's Paradox]
- Distinguish between classes that cause paradoxes to occur vs the rest

### Definitions
- Classes considered to be primitive notion in NBG set theory
- ***Set*:** $A$ is a set $\iff\exists B \ni A \epsilon B$
