- A Turing Machine is the mathematical abstraction of what a computer is
- At it's core a computer can take input, store information in memory, process the information in the memory and then return an output
- A computer architecture essentially is the practical implementation of a Turing machine using digital circuits
- The commonly adopted architecture that most computers follow is called the Von-Neumann Architecture
